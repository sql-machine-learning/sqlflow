# SQLFlow Demo

Before you start, you need to set up test MySQL server as described [here](https://github.com/wangkuiyi/sqlflow/tree/develop/example/datasets)

Start SQLFlow as
```golang
go get -d ./... && go run main.go -logdir="/path/to/logs/" -loglevel="info"
```

#### Regular SQL Statements

```sql
sqlflow> SHOW DATABASES;
+--------------------+
| Database           |
+--------------------+
| churn              |
| information_schema |
| iris               |
| mysql              |
| performance_schema |
| sqlflow_models     |
| sqlfs              |
| sys                |
+--------------------+
sqlflow> SELECT * FROM iris.iris LIMIT 1;
+--------------+-------------+--------------+-------------+-------+
| sepal_length | sepal_width | petal_length | petal_width | class |
+--------------+-------------+--------------+-------------+-------+
|          6.4 |         2.8 |          5.6 |         2.2 |     2 |
+--------------+-------------+--------------+-------------+-------+
```

#### Train

```sql
sqlflow> SELECT *
FROM iris.iris
TRAIN DNNClassifier
WITH
  n_classes = 3,
  hidden_units = [10, 20]
COLUMN sepal_length, sepal_width, petal_length, petal_width
LABEL class
INTO my_dnn_model;
-----------------------------
2018/12/16 15:03:54 tensorflowCmd: run in Docker container
Job success
```

#### Predict

```sql
sqlflow> SELECT *
FROM iris.iris
PREDICT iris.predict.class
USING my_dnn_model;
-----------------------------
2018/12/16 15:05:58 tensorflowCmd: run in Docker container
Job success
```
